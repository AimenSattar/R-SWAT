SWAT project

  • Extract output from "watout.dat" FileType: You can read SWAT outputs from multiple 'watout.dat' FileType with different FileName, add more rows if you do so. The Column specifies which column(s) from that file you want to extract (e.g., column 4 and 5 and 7) -> the input should be '4,5,7' with comma to seperation. Please leave the Reach column empty.
  • In case of reading from output.rch, output.sub, output.hru, the Reach number is also means the subbasin or hru number. For example,
    • You want to read from column 7 reach number 1, 2
    • and from columnn 8 reach number 1, 2, 3
    • → Input to the Column should be '7, 8', to the Reach should be
      '1, 2 * 1, 2, 3 * 3'
  • If you want to read from other files, please modified the source code file ./R/userReadSwatOutput.R, more information about this could be found in the Section 4 of R-SWAT wiki page https://github.com/tamnva/R-SWAT/wiki/R-SWAT-User-Manual
  • Please check the tick box below (Display corresponding observed file names) to ensure that all numbers you enter here appear in the display table. The information from this text box is very important, as the observed data files should have the same name as listed in here

SWAT+ project

  • Currently, R-SWAT can extract output from different channel files (e.g., channel_sd_day.txt, channel_sd_month.txt, and so on). The Column number indicates which columns in this output file you want to extract and the Reach number indicates which reaches you want to extract. For example, if you want to extract from column 48 and for reach numbers 1 and 2, and column 53 for reach numbers 1,2,3. You write in the Column '48, 53' and in the Reach '1, 2 * 1, 2, 3'
  • You can also read from other files by modifying the source code file ./R/userReadSwatOutput.R
