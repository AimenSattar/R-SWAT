Short Intro

  • Here you can calibrate your SWAT/SWAT+ model manually and the model outputs will be visualized directly here.
  • This could be used for teaching purpose, e.g., to understand the effects of different processes/parameter values on targeted output variables (e.g. streamflow)

Quick start!

  • To run this, the following steps need to be done beforehand
    • 1. General setting
    • 2. Parameter sampling (only subsection 1)
    • 3. Run SWAT (only subsections 1 and 2)
    • 4.1. Objective functions (only subsection 1 and 2)

How does this section connect to other sections!

  • 1. Select parameter value takes parameters from 2. Parameter sampling (the parameter table in subsection 1). Every time you change something in this parameter table, the parameter name/range in this section will be automatically updated.
  • 2. Model runs will run the model with:
    • General project information from 1. General setting,
    • Parameter values from 1. Select parameter value
    • Model output variables from 3. Run SWAT (subsection 1)
    • Date range of model outputs from 3. Run SWAT (only subsection 2)
  • 3. Plot simulated variables will plot the output variables defined in 3. Run SWAT (subsection 1) and observed data defined in 4.1 Objective function (only subsection 2)
