SWAT project → Select 'swatParam.txt' file

  • This file is provided along with this tool, it can be extracted to your local folder by typing "extracExampleData(exampleData, "swatParam", tempdir())" in R or can be downloaded from here.
  • This file contains information about the location SWAT parameter in SWAT input file and its format as well as its theoretical range.
  • For example (the HRU_FR parameter in all .hru files is located at line 2, column 1 to 16 with a precision of 7 digits after the decimal.
  • The columns AbsoluteMin and AbsoluteMax in this files means the absolute min and max, R-SWAT will check and make sure that updated parameter values will always be within this range (if the updated values out of this range, e.g., higher than the AbsoluteMax, R-SWAT will use the AbsoluteMax , if the updated values smaller than AbsoluteMin, R-SWAT will use the AbsoluteMin.
  • Please check if the AbsoluteMin and AbsoluteMax for the parameters that you are going to calibrate/change are reasonable.
  • If there is a parameter that is missing, simply add it to this file at any line (using text editor to open the file and edit manually) and then load to R-SWAT
