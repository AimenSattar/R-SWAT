Please select the date range for sensitivity analysis/calibration. The min and maximum dates are automatically detected by R-SWAT by reading information from the file.cio file in case of SWAT or time.sim and print.prt files in case of SWAT+ in the TxtInOut folder. Feel free to select any dates within the given range of R-SWAT

SWAT project

  • Currently, only reading daily output is possible. To read monthly, yearly or other output time steps, please use the option "userReadSwatOutput". In this case, the date range of extraction will not have any effect since it is defined by the user within the "userReadSwatOutput" option

SWAT+ project

  • When user select "userReadSwatOutput" option, the date range defined in here will not have any effect.
  • When reading from the monthly outputs (e.g., channel_sd_mon.txt, channel_sdmorph_mon.txt) only the month and year selected in here will be taken (e.g., if user selects extracting output from 01 Jan 2000 or from 31 Jan 2000, the program will extract from Jan, 2000).
  • When reading from the monthly outputs (e.g., channel_sd_yr.txt, channel_sdmorph_yr.txt) only the year selected in here will be taken
